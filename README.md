# Get TON Mining Bot

Bot Telegram mining TON berbasis Python.

## Fitur:
- Mining klik setiap 1 menit
- Saldo bertambah otomatis
- Simpan data sementara di RAM

## Deploy:
1. Upload ke GitHub atau Replit
2. Set environment variable:
   - BOT_TOKEN = <TOKEN_BOT_KAMU>
3. Jalankan `main.py`
